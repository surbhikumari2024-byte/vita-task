**VitaTask** is a modern, single-page productivity application designed for students and professionals. Built with a focus on speed, aesthetics, and user experience, it helps users manage their assignments and daily goals efficiently.

## ✨ Features

-   **Dashboard Overview**: Real-time stats showing total tasks, completed items, and daily streaks.
-   **Task Management**: Easily add, toggle completion status, and delete tasks.
-   **Persistent Storage**: Uses `LocalStorage` so your tasks remain even after refreshing the page.
-   **Modern UI/UX**: 
    -   Dark-themed interface powered by **Tailwind CSS**.
    -   Responsive sidebar navigation.
    -   Smooth hover animations and transitions.
-   **Dynamic Date**: Automatically displays the current date for better daily planning.

## 🛠️ Tech Stack

-   **HTML5**: Semantic structure.
-   **Tailwind CSS**: Utility-first styling for a sleek, modern look.
-   **Vanilla JavaScript (ES6+)**: Core logic, DOM manipulation, and data persistence.
-   **Font Awesome**: High-quality vector icons.
